### nosleeping-bot
 Nosleeping bot for discord

Be sure to fill in `config/default.json` before running the bot.
Do not edit `datastore/users.db`. Screwing with the values while the bot is running can cause issues.

## How to install:
From the project directory run
```
npm install
```
After package installation run
```
npm start
```

You should see the bot connect in the terminal with a link. Click the link to add the bot to your server.
